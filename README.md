
========================================
使い方
========================================
1. このHTMLファイルをブラウザで開くだけで動作します（ビルド不要）
2. プリセットボタン（Pomodoro / 短め集中 / カスタム）を選択
3. スタートボタンを押すとタイマーが開始されます
4. キーボード操作: Space=スタート/停止、R=リセット、1/2/3=プリセット選択
5. 設定は自動保存され、次回ページを開いた際に復元されます

========================================
実装上の主要なポイント
========================================
1. **メモリ内状態管理とローカルストレージの使い分け**
   - タイマーの実行状態（残り時間、動作中フラグ）はメモリ内で管理
   - ユーザー設定（プリセット選択、カスタム値）のみをlocalStorageに保存
   - try-catchでストレージエラーをハンドリングし、フォールバック表示

2. **SVGとCSSアニメーションによる可愛い演出**
   - プログレスリングはSVGのstroke-dashoffsetで実装
   - 小さなキャラクターをSVGで描画し、タイマー進行で表情変化
   - 完了時にcelebrate keyframeアニメーションで祝福演出
   - Web Audio APIで軽量なベル音を動的生成（外部ファイル不要）

3. **アクセシビリティとキーボード操作**
   - 全ボタンにaria-label属性を設定
   - キーボードフォーカス時に視覚的なアウトライン表示
   - Space/R/数字キーでの操作に対応（入力フォーカス時は無効化）
   - レスポンシブデザインでモバイルでも快適に操作可能

========================================
外部素材について
========================================
このタイマーは外部素材を一切使用していません。
- フォント: システムフォント（Segoe UI / Yu Gothic UI / Meiryo）
- アイコン: インラインSVGで描画
- サウンド: Web Audio APIで動的生成
- すべての装飾はCSS/SVGで実装

ライセンス: このコードはCC0（パブリックドメイン）として自由に利用可能です
